USLCC Error ChartR Documentation

USLCC Data for Analysis

Description

This dataset represents the Error Master Chart from the USLCC Center. Each observation represents one error event that occurred during the process of Radiation Oncology following the treatment and is identified by a unique ID. Each observation includes some collected data in the form of a free text field, an identification date, the role of the identifying individual, the treatment intent, and the treatment method. The category classifies each observation that the error event falls into, represented by the BroadCat variable. BroadCat is a collapsed super category into four levels from the original ten levels of the event category and represents which part of the radiotherapy process the error occurred in. For the sake of proper statistical analysis, the records with a super-category of "Treatment" have been removed as there were too few records to provide any predictive power. This data was consolidated by RadPhysics Services LLC and cleaned by NCSU ST495 Team B to create a classification model that will classify an error event by the Error Category that it should fall into, using the features included in the dataset.

Format

A data frame with 285 observations on the following 11 variables.

ID

The unique ID for the error event.

BroadCat

The Error Category collapsed into one of three super-categories as described in "Automated Error Labeling in Radiation Oncology via Statistical Natural Language Processing".

IDbyRole

Role(s) of the person who identified and input the deviation and their letter convention.

DateTime

Date and time when the attribute was selected by the user.

Quarter

The quarter of the year when the attribute was selected by the user.

Month

The month of the year when the attribute was selected by the user.

TXIntent

User is asked the following regarding the treatment intent: 0 = NA, 1 = Curative, 2 = Palliative (pain relief only).

TXMethod

User is asked what kind of treatment method is being utilized: 0 = NA, 1 = Conventional EBRT, 2 = 3D-CRT, 3 = IMRT, 4 = SRS, 5 = SBRT.

Status

Ranges from 7 to 10.

DevType

Main type of deviation selected by the user on the first screen when entering an error: 0 = Clinical, 1 = Radiation Safety, 2 = Quality Assurance.

Desc

After an attribute (deviation) is selected, this is the free text description input by the user describing the details of the deviation.
